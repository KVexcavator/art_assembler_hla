Модули стандартной библиотеки HLA
---------------------------------------------
args        Подпрограммы поддержки разбора параметров командной строки.
arrays      Объявления и операции с массивами.
bits        Функции обработки битов.
blobs       Большие двоичные объекты — операции с большими блоками двоичных данных.
bsd         Вызовы API ОС для FreeBSD (только версия HLA FreeBSD).
chars       Операции с символьными данными.
console     Операции переносимой консоли (текстовый экран) (перемещение курсора, очистка экрана и т. д.).
conv        Различные преобразования между строками и другими значениями.
coroutines  Поддержка сопрограмм («кооперативная многозадачность»).
cset        Функции набора символов.
DateTime    Функции календаря, даты и времени.
env         Доступ к переменным среды ОС.
excepts     Подпрограммы обработки исключений.
fileclass   Объектно-ориентированный ввод и вывод файлов.
fileio      Подпрограммы ввода и вывода файлов.
filesys     Доступ к файловой системе ОС.
hla         Специальные константы HLA и другие значения.
Linux       Системные вызовы Linux (только версия HLA Linux).
lists       Класс HLA для управления связанными списками.
mac         Вызовы API Mac OS (только версия HLA для Mac OS X).
math        Арифметика повышенной точности, трансцендентные функции и другие математические функции.
memmap      Операции с файлами, отображаемыми в памяти.
memory      Выделение памяти, освобождение памяти и код поддержки.
patterns    Библиотека сопоставления шаблонов HLA.
random      Генераторы псевдослучайных чисел и код поддержки.
sockets     Набор функций и классов сетевой связи.
stderr      Предоставляет пользовательский вывод и несколько других функций поддержки.
stdin       Процедуры пользовательского ввода.
stdio       Модуль поддержки для stderr, stdin и stdout.
stdout      Предоставляет пользовательский вывод и несколько других функций поддержки.
strings     Мощная библиотека строк HLA.
tables      Процедуры поддержки таблиц (ассоциативных массивов).
threads     Поддержка многопоточных приложений и синхронизации процессов.
timers      Поддержка событий синхронизации в приложении.
win32       Константы, используемые в вызовах Windows (только версия HLA для Windows).
x86         Константы и другие элементы, специфичные для процессора 80x86.
